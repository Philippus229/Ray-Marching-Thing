# Ray-Marching-Thing
A python script for visualizing thee-dimensionsal fractals and basic shapes using ray marching.

## Usage
Just uncomment the object(s) you wanna see or add your own (in sceneSDF2) and run the script.

## Requirements
- Python 3.x
- Pygame
- PyGLM
- NumPy
